## Format Rules

- Each GBP post uses the format in `07-ASSETS/templates/gbp-post-template.md`.
- Each GBP post extracts one angle from the source blog. It does not summarize the full post.
- The GBP headline cannot repeat the blog title.
- The GBP opening cannot repeat the blog opening.
- Use present tense, active voice, and fewer than 300 words.
- Include one CTA label at the end.
- Link each post back to the source blog once it is live.
