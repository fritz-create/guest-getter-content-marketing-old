# Content Marketing Template

This is a reusable content marketing workspace for restaurant brands. It is currently a master template, not an active client folder.

## Start Here

For every session, read these files first:

1. `AGENT.md` - full workflow rules and phase gates.
2. `CLAUDE.md` - Claude-specific startup instructions.
3. `session-notes.md` - shared cross-LLM handoff notes.
4. `00-START-HERE/SESSION-STATE.md` - formal phase status.
5. `00-START-HERE/CLIENT-INTAKE.md` - client website URL and notes.

If `CLIENT-INTAKE.md` has no website URL, do not start phase work. This means the folder is still the master template.

## Core Rule

Do not advance phases, create client content, or publish-ready assets without the approvals required in `AGENT.md`.

## Main Workflow

1. Website research
2. Competitor research
3. Brand interview
4. Brand documents
5. Content pillars and topical map
6. Calendar backlog
7. Blog, GBP, and repurposing production

## Handoff

Use `00-START-HERE/SESSION-STATE.md` for formal phase tracking. Use `session-notes.md` for practical handoff context, recent decisions, blockers, and template changes.
